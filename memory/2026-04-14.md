**3. Missing Contextual Analysis as Named Failure Mode in Analytical Reports (April 13 PM)**

**4. Consensus vs. Execution Mode — The AI-Speed Communication Diagnostic (April 14 AM)**
HBR (April 7, 2026): consensus-based decision-making is a liability in AI-speed operating environments. When organizations shift from consensus to directed authority, the communication requirements change fundamentally. Consensus documents lead with context and evidence, build to a recommendation. Execution documents lead with the decision, then explain logic and path. Document mode mismatch (execution content in consensus structure) is a named editing problem — the most common error in post-decision executive communication. Diagnostic: is this document trying to build consensus (pre-decision) or activate execution (post-decision)? For LinkedIn: posts that frame problems from "here's what to consider" lose reach compared to posts starting from a decided position.
IMfA nominated: YES.
